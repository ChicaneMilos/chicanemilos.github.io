Take Next Target

The game features dynamic gameplay with unique characters, abilities, and power-ups, supported by Mirror’s server architecture for multiplayer functionality and Blender for visual assets.

Classroom Scheduler

Web application developed for the Academy of Applied Technical and Preschool Studies in Niš, Serbia. Built with HTML, JavaScript, and .NET (ASP.NET, C#), and hosted on Azure. Designed for managing and scheduling academic activities, including lectures and exams, with graphical seating arrangements.

Quiz Game

Quiz Game is a codename for a quiz game currently in development using Unity and C#, used by the Academy of Applied Technical and Preschool Studies in Niš, Serbia. The game aims to leverage VR technology in the future to provide an engaging educational experience and test participants' knowledge during promotional events.

Parking Simulation

The Parking Simulation project is a comprehensive JavaScript application with robust backend support in both Python and .NET (ASP.NET). It effectively simulates a detailed parking environment, allowing users to manage vehicle data, track various attributes, and accurately calculate parking fees based on duration.

About Me

Hello! I’m Miloš, and my interest in technology has been a constant since childhood. My journey into programming began in 2017 when the online servers for one of my favorite games were shut down. This event ignited my desire to create my own game, leading me to dive into Unity and C# and diligently follow tutorials to learn video game development. Through persistent effort over the years, I have developed expert knowledge in video game development with Unity.

Throughout my learning journey, I expanded my skill set to include 3D modeling with Blender and SQL to meet game development needs. During my academic studies, I further broadened my expertise by exploring .NET technologies and C# beyond Unity, incorporating ASP.NET for backend development, and enhancing my proficiency in JavaScript and Python.

Today, I am actively working on my most innovative and complex Unity game with my team, developing practical applications for my college, and continuously striving to enhance my knowledge and skills. My passion for technology drives me to embrace new challenges and advance my expertise every day.
